版本
```shell
docker --version

docker-compose --version

# Docker 20.10 版本开始
docker compose version
```

镜像（镜像源太少，会出现错误）
```shell
vi /etc/docker/daemon.json

{
  "registry-mirrors": [
    "https://mirror.aliyun.com",  # 这是错误的，需要登陆阿里云去复制专属与自己的镜像
    "https://docker.mirrors.ustc.edu.cn",
    "https://hub-mirror.c.163.com"
  ]
}

sudo systemctl daemon-reload
sudo systemctl restart docker

```


重启 Docker
```shell
sudo systemctl daemon-reload
sudo systemctl restart docker
```

启动所有容器(需要进到有docker-compose.yml的目录里)
```shell
docker compose up -d
```

启动某个容器
```shell
docker compose up -d 容器名

# 例如
docker compose up -d mysql57
docker compose up -d mysql80
```

重启容器
```shell
docker compose restart
```

进入到容器里面(需要进到有docker-compose.yml的目录里)
```shell
docker compose exec mysql80 bash
```

检查配置
```shell
docker compose config
```

整台 CentOS 主机上所有 Docker 容器
```shell
docker ps -a
```

整台 CentOS 主机上运行中的容器
```shell
docker ps
```

查看当前容器的运行状态
```shell
docker compose ps
```
